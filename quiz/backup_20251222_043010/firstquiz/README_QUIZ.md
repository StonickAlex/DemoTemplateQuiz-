# Интерактивный квиз "Узнай, какая суперсила передалась тебе по роду"

## Описание

Простой HTML-квиз с минималистичным дизайном для прохождения теста о связи с родом и сверхспособностях.

## Файлы

- `quiz.html` - основной HTML-файл с квизом
- `quiz_data.json` - данные квиза (вопросы, ответы, баллы, результаты)
- `generate_quiz_json.py` - скрипт для генерации JSON из markdown файла

## Как использовать

### Вариант 1: Просто открыть в браузере

1. Убедитесь, что файлы `quiz.html` и `quiz_data.json` находятся в одной папке
2. Откройте `quiz.html` в браузере (двойной клик или через File → Open)

### Вариант 2: Через локальный сервер (рекомендуется)

```bash
# Python 3
python3 -m http.server 8000

# Или Python 2
python -m SimpleHTTPServer 8000

# Или через Node.js (если установлен)
npx http-server
```

Затем откройте в браузере: `http://localhost:8000/quiz.html`

## Обновление данных квиза

Если вы изменили `QuizKatrin_IMPROVED.md`, запустите:

```bash
python3 generate_quiz_json.py
```

Это создаст/обновит файл `quiz_data.json`.

## Особенности

- **Простой и строгий дизайн** - черно-белая цветовая схема
- **Адаптивная система** - вопросы задаются в зависимости от ответов
- **Отслеживание баллов** - текущий счет отображается в правом верхнем углу
- **Прогресс-бар** - показывает прогресс по основным вопросам
- **Результаты** - автоматический расчет уровня связи с родом

## Технические детали

- Чистый JavaScript (без библиотек)
- Работает в современных браузерах
- Адаптивный дизайн для мобильных устройств
- Локальное хранение не используется (можно добавить при необходимости)
